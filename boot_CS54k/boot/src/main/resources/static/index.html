社团简介

成立背景

黑河学院软件项目社团成立于2021年，是一个由电脑爱好者组成的学生社团。社团由杨文君老师指导，以计算机软件开发和发语言模型研究为主要研究方向。

我们以"奉献、友爱、互助、进步"为宗旨，进行电脑知识学习与实践。

社团宗旨

黑河学院软件开发社团自秉承着培养未来软件开发精英的使命。我们致力于打造一个充满活力、创新和协作精神的技术社区。

通过实际项目开发、技术研讨和团队合作，让学生在实践中学习和成长。

社团成果

边疆智能计算 PFAI

2024年4月

我们成功自主研发了PFAI（蛋白质功能分析工具），这是我们在蛋白质相关功能研究领域取得的重要成果。

收集了248万条人类目前已知的蛋白质数据，使用54万条人类已发表相关论文文献且已对该蛋白质序列的蛋白质作为数据集。

2024CNCC中国计算机大会

2024年

东北地区受资助名单，黑龙江省仅八个人获得资助，我社团由于优秀的人工智能前沿课题，获得三个资助名额。

校园高清正射影片

2023年

黑河学院后勤处采纳黑河学院鸟瞰图及其正射影像，为校园规划和管理提供了重要数据支持。

人工智能公务员智慧小花

2023年

为黑河市花园街道搭建人工智能公务员"智慧小花"，提升了基层政务服务的智能化水平。

黑河学院3D数字校园

2023年

为迎接新生扫描黑河学院整体建筑，搭建web端、手机端、电脑端3D客户端，计划为后勤部门提供数字化校园解决方案。

DeepSeek校园部署

2024年

为黑河学院部署DeepSeek人工智能平台，获得了从校长到学生的广泛支持，推动了校园智能化建设。
