社团发展历程

2021年
社团初创

黑河学院软件项目社团正式成立，由杨文君老师指导，最初由一群计算机爱好者组成。我们以"奉献、友爱、互助、进步"为宗旨，进行电脑知识学习与实践。

初创阶段 计算机基础
2022年
组织架构完善

社团组织结构进一步完善，设立了外联部、前端开发部、后端开发部、项目测试部等部门，为后续发展奠定基础。

组织建设 部门分工
2023年
首次参赛与技术突破

社团首次参加省级软件设计大赛，获得优秀奖，开启了社团参赛历程。同时，我们建立了第一个服务器基础设施。

首次参赛 服务器建设
2023年
人工智能方向确立

成立人工智能研究部，开始专注于大语言模型和计算机视觉研究。为黑河市花园街道搭建人工智能公务员"智慧小花"，并完成黑河学院3D数字校园项目。

AI研究 3D建模
2023年
重大科研突破

自主研发PFAI(蛋白质功能分析工具)，获得CNCC中国计算机大会资助。东北地区受资助名单中，黑龙江省仅八个人获得资助，我社团获得三个资助名额。

PFAI CNCC资助
2024年
国际合作与平台建设

与俄罗斯高校建立合作关系，共同开展大模型研究项目。为黑河学院部署DeepSeek人工智能平台，获得了从校长到学生的广泛支持。

中俄合作 DeepSeek
未来规划
边疆人工智能中心

两年计划：以软开为基础，组建黑河学院人工智能服务器中心。三年计划：组建边疆人工智能中心，推动区域人工智能技术发展。
